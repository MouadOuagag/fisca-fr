# Cotisations TNS 2026 — Artisans, commerçants et libéraux

Les **travailleurs non-salariés (TNS)** relèvent du régime social des indépendants (SSI, intégré à la Sécurité sociale). Leurs cotisations sont calculées sur le **bénéfice net** de l'année N-2 (avec régularisation en N+1).

## Les taux de cotisations 2026

### Artisans et commerçants (SSI)

| Risque | Taux | Base de calcul |
|--------|------|---------------|
| Maladie-maternité (indemnités) | 0,85 % | Revenu < 18 547 € : taux réduit progressif |
| Maladie-maternité (prestations) | 6,5 % | Revenu au-delà de 18 547 € |
| Retraite de base | 17,75 % | Revenu ≤ 47 100 € (PASS) |
| Retraite de base (au-delà du PASS) | 0,60 % | Revenu > 47 100 € |
| Retraite complémentaire | 7 % | Revenu ≤ 42 946 € |
| Retraite complémentaire tranche 2 | 8 % | De 42 946 € à 188 400 € |
| Invalidité-décès | 1,3 % | Revenu ≤ 47 100 € |
| Allocations familiales | 0 % ou 3,1 % | 0 % si revenu < 47 100 €, dégressif jusqu'à 1,4 × PASS |
| CSG/CRDS | 9,7 % | Revenu + cotisations obligatoires |
| Formation professionnelle | 0,29 % | PASS (47 100 €) |

**Taux global moyen estimé : 40 à 45 % du bénéfice net**

### Professions libérales (CIPAV — pour les professions relevant de la CIPAV)

| Risque | Taux | Base de calcul |
|--------|------|---------------|
| Maladie-maternité | Identique au SSI | — |
| Retraite de base | 17,75 % | ≤ PASS |
| Retraite complémentaire (classe A à H) | Variable selon classe | — |
| Invalidité-décès | 1,3 % | ≤ PASS |
| Allocations familiales | 0 % à 3,1 % | Selon revenu |
| CSG/CRDS | 9,7 % | — |

> Depuis 2020, la plupart des nouvelles professions libérales rejoignent directement le SSI (ex-RSI), sauf les professions réglementées historiquement affiliées à la CIPAV (architectes, experts-comptables, etc.).

## Le calcul des cotisations provisionnelles

Les cotisations sont calculées en **deux temps** :

1. **Cotisations provisionnelles (N)** : basées sur le revenu N-2
2. **Régularisation (N+1)** : ajustement sur le revenu réel N

**1re année d'activité** : cotisations forfaitaires provisionnelles sur une base minimale

| Activité | Base forfaitaire 1re année | Cotisations estimées |
|---------|--------------------------|---------------------|
| Artisan/commerçant | 19 % du PASS (8 949 €) | ~3 500 à 4 000 € |
| Profession libérale SSI | 19 % du PASS | ~3 500 à 4 000 € |

## Le minimum de cotisations

Même sans bénéfice, des **cotisations minimales** sont dues (pour maintenir les droits à la retraite et à la maladie) :

| Cotisation | Montant 2026 |
|-----------|-------------|
| Retraite de base | ~1 100 € |
| Maladie-maternité | ~0 € (exonération totale sous seuil) |
| Invalidité-décès | ~70 € |
| Formation professionnelle | ~137 € |

## Cotisations Madelin et PER

Les TNS peuvent déduire de leur bénéfice imposable :
- Les cotisations **Madelin** (contrats retraite, prévoyance, santé)
- Les versements sur un **PER individuel**

**Plafond de déduction PER :** 10 % du bénéfice net (dans la limite de 8 × PASS) soit jusqu'à **37 680 €** en 2026.

## Optimisation : rémunération vs dividendes

| Critère | Rémunération TNS | Dividendes SAS |
|---------|-----------------|----------------|
| Cotisations sociales | ~45 % | 0 % (SAS) |
| Déductibilité fiscale | Oui (IS) | Non |
| Protection sociale | Complète | Limitée |
| Retraite | Droits élevés | Pas de droits |
